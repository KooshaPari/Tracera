# Workflows

The TraceRTM Go SDK provides comprehensive workflow management capabilities with idiomatic Go patterns, allowing you to define, configure, and execute status transitions for your traceability items.

## Overview

Workflows in TraceRTM define:

- **States**: The possible statuses an item can have (e.g., Draft, In Review, Approved)
- **Transitions**: The allowed movements between states (e.g., Submit for Review, Approve, Reject)
- **Rules**: Conditions that must be met for transitions (e.g., required fields, approvals)
- **Actions**: Automated actions triggered by transitions (e.g., notifications, field updates)

## Listing Workflows

### Get Project Workflows

```go
package main

import (
    "context"
    "fmt"
    "log"
    "os"

    "github.com/tracertm/tracertm-go"
)

func main() {
    client := tracertm.NewClient(os.Getenv("TRACERTM_API_KEY"))
    ctx := context.Background()

    // List all workflows in a project
    workflows, err := client.Workflows.List(ctx, &tracertm.ListWorkflowsInput{
        ProjectID: "proj_abc123",
    })
    if err != nil {
        log.Fatal(err)
    }

    for _, workflow := range workflows.Data {
        fmt.Printf("%s (%s)\n", workflow.Name, workflow.ID)
        fmt.Printf("  Item types: %v\n", workflow.ItemTypes)
        fmt.Printf("  States: %d\n", len(workflow.States))
        fmt.Printf("  Transitions: %d\n", len(workflow.Transitions))
    }
}
```

### Get Workflow Details

```go
// Get a specific workflow with full details
workflow, err := client.Workflows.Get(ctx, "wf_abc123")
if err != nil {
    log.Fatal(err)
}

fmt.Printf("Workflow: %s\n", workflow.Name)
fmt.Printf("Description: %s\n", workflow.Description)
fmt.Printf("Default: %v\n", workflow.IsDefault)

// List states
fmt.Println("\nStates:")
for _, state := range workflow.States {
    fmt.Printf("  - %s (%s)\n", state.Name, state.ID)
    fmt.Printf("    Category: %s\n", state.Category) // initial, in_progress, done
    fmt.Printf("    Color: %s\n", state.Color)
}

// List transitions
fmt.Println("\nTransitions:")
for _, transition := range workflow.Transitions {
    fmt.Printf("  - %s\n", transition.Name)
    fmt.Printf("    From: %s -> To: %s\n", transition.FromState, transition.ToState)
}
```

## Transitions

### Get Available Transitions

```go
// Get transitions available for a specific item
transitions, err := client.Items.AvailableTransitions(ctx, "item_req_001")
if err != nil {
    log.Fatal(err)
}

fmt.Println("Available transitions:")
for _, t := range transitions {
    fmt.Printf("  - %s (%s)\n", t.Name, t.ID)
    fmt.Printf("    To: %s\n", t.ToState)
    if len(t.RequiredFields) > 0 {
        fmt.Printf("    Required: %v\n", t.RequiredFields)
    }
    if len(t.RequiredPermissions) > 0 {
        fmt.Printf("    Permissions: %v\n", t.RequiredPermissions)
    }
}
```

### Execute Transition

```go
// Transition an item to a new state
item, err := client.Items.Transition(ctx, "item_req_001", &tracertm.TransitionInput{
    Transition: "submit_for_review",
    Comment:    "Ready for technical review",
})
if err != nil {
    log.Fatal(err)
}

fmt.Printf("New status: %s\n", item.Status)
```

### Transition with Fields

```go
// Some transitions require additional fields
item, err := client.Items.Transition(ctx, "item_req_001", &tracertm.TransitionInput{
    Transition: "approve",
    Comment:    "Approved after technical review",
    Fields: map[string]interface{}{
        "approved_by":    "user_123",
        "approval_date":  "2025-01-15",
        "approval_notes": "All acceptance criteria met",
    },
})
```

### Transition with Assignee Change

```go
// Transition and reassign in one operation
item, err := client.Items.Transition(ctx, "item_req_001", &tracertm.TransitionInput{
    Transition: "assign_to_development",
    AssigneeID: "user_dev_001",
    Comment:    "Assigned to development team",
})
```

## Bulk Transitions

### Transition Multiple Items

```go
// Transition multiple items at once
results, err := client.Items.BulkTransition(ctx, &tracertm.BulkTransitionInput{
    ItemIDs:    []string{"item_req_001", "item_req_002", "item_req_003"},
    Transition: "approve",
    Comment:    "Bulk approval after sprint review",
})
if err != nil {
    log.Fatal(err)
}

for _, result := range results {
    if result.Success {
        fmt.Printf("%s: Transitioned to %s\n", result.ItemID, result.NewStatus)
    } else {
        fmt.Printf("%s: Failed - %s\n", result.ItemID, result.Error)
    }
}
```

### Conditional Bulk Transition

```go
// Transition items matching criteria
results, err := client.Items.BulkTransitionByFilter(ctx, &tracertm.BulkTransitionByFilterInput{
    ProjectID: "proj_abc123",
    Filter: &tracertm.ItemFilter{
        Type:       "requirement",
        Status:     "in_review",
        AssigneeID: "user_123",
    },
    Transition: "approve",
    Comment:    "Approved all reviewed requirements",
})
if err != nil {
    log.Fatal(err)
}

fmt.Printf("Transitioned %d items\n", results.SuccessCount)
fmt.Printf("Failed: %d\n", results.FailureCount)
```

## Workflow Configuration

### Create Workflow

```go
// Create a new workflow
workflow, err := client.Workflows.Create(ctx, &tracertm.CreateWorkflowInput{
    ProjectID:   "proj_abc123",
    Name:        "Agile Requirement Workflow",
    Description: "Workflow for agile requirement management",
    ItemTypes:   []string{"requirement", "user_story"},
    States: []tracertm.WorkflowState{
        {Name: "Backlog", Category: "initial", Color: "#gray"},
        {Name: "In Analysis", Category: "in_progress", Color: "#blue"},
        {Name: "Ready for Dev", Category: "in_progress", Color: "#yellow"},
        {Name: "Done", Category: "done", Color: "#green"},
    },
    Transitions: []tracertm.WorkflowTransition{
        {Name: "Start Analysis", FromState: "Backlog", ToState: "In Analysis"},
        {Name: "Analysis Complete", FromState: "In Analysis", ToState: "Ready for Dev"},
        {Name: "Mark Done", FromState: "Ready for Dev", ToState: "Done"},
        {Name: "Reopen", FromState: "Done", ToState: "Backlog"},
    },
})
if err != nil {
    log.Fatal(err)
}

fmt.Printf("Created workflow: %s\n", workflow.ID)
```

## Transition Rules

### Define Validation Rules

```go
// Add validation rule to transition
err := client.Workflows.AddTransitionRule(ctx, "wf_abc123", "trans_approve", &tracertm.TransitionRuleInput{
    RuleType: "field_required",
    Config: map[string]interface{}{
        "fields": []string{"acceptance_criteria", "priority"},
    },
})

// Add permission rule
err = client.Workflows.AddTransitionRule(ctx, "wf_abc123", "trans_approve", &tracertm.TransitionRuleInput{
    RuleType: "permission_required",
    Config: map[string]interface{}{
        "permissions": []string{"approve_requirements"},
    },
})

// Add condition rule
err = client.Workflows.AddTransitionRule(ctx, "wf_abc123", "trans_approve", &tracertm.TransitionRuleInput{
    RuleType: "condition",
    Config: map[string]interface{}{
        "expression":    "item.linked_test_cases > 0",
        "error_message": "Item must have at least one linked test case",
    },
})
```

### Check Transition Validity

```go
// Check if a transition is valid for an item
validity, err := client.Items.CheckTransition(ctx, "item_req_001", "approve")
if err != nil {
    log.Fatal(err)
}

if validity.Valid {
    fmt.Println("Transition is allowed")
} else {
    fmt.Println("Transition blocked:")
    for _, reason := range validity.BlockingReasons {
        fmt.Printf("  - %s\n", reason)
    }
}
```

## Error Handling

```go
import "errors"

item, err := client.Items.Transition(ctx, "item_req_001", &tracertm.TransitionInput{
    Transition: "approve",
})

if err != nil {
    var transitionErr *tracertm.TransitionNotAllowedError
    var validationErr *tracertm.WorkflowValidationError
    var requiredFieldErr *tracertm.RequiredFieldMissingError

    switch {
    case errors.As(err, &transitionErr):
        fmt.Printf("Transition not allowed: %s\n", transitionErr.Message)
        fmt.Printf("Current status: %s\n", transitionErr.CurrentStatus)
        fmt.Printf("Blocking reasons: %v\n", transitionErr.BlockingReasons)

    case errors.As(err, &requiredFieldErr):
        fmt.Printf("Missing required fields: %v\n", requiredFieldErr.MissingFields)

    case errors.As(err, &validationErr):
        fmt.Printf("Workflow validation failed: %s\n", validationErr.Message)

    default:
        fmt.Printf("Error: %v\n", err)
    }
}
```

## Concurrency Patterns

### Bulk Process with Worker Pool

```go
import "sync"

func bulkTransition(ctx context.Context, client *tracertm.Client, itemIDs []string, transition string, workers int) {
    var wg sync.WaitGroup
    sem := make(chan struct{}, workers)

    for _, itemID := range itemIDs {
        wg.Add(1)
        go func(id string) {
            defer wg.Done()
            sem <- struct{}{}
            defer func() { <-sem }()

            _, err := client.Items.Transition(ctx, id, &tracertm.TransitionInput{
                Transition: transition,
            })
            if err != nil {
                log.Printf("Failed to transition %s: %v", id, err)
            } else {
                log.Printf("Transitioned: %s", id)
            }
        }(itemID)
    }

    wg.Wait()
}
```

### Context Cancellation

```go
func transitionWithTimeout(client *tracertm.Client, itemID, transition string) (*tracertm.Item, error) {
    ctx, cancel := context.WithTimeout(context.Background(), 30*time.Second)
    defer cancel()

    return client.Items.Transition(ctx, itemID, &tracertm.TransitionInput{
        Transition: transition,
    })
}
```
