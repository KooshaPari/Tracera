# Artifacts

The TraceRTM JavaScript SDK provides comprehensive support for managing file artifacts such as documents, images, test results, and other files associated with your traceability items.

## Overview

Artifacts are files attached to items or projects in TraceRTM. Common use cases include:

- Attaching specification documents to requirements
- Uploading test reports to test cases
- Storing design mockups with user stories
- Associating code snippets or logs with bugs

---

## Uploading Artifacts

### Node.js - File Path

```typescript
import { TraceRTMClient } from '@tracertm/sdk';
import { createReadStream } from 'fs';
import path from 'path';

const client = new TraceRTMClient();

// Upload a file from disk
const artifact = await client.artifacts.upload({
  itemId: 'item_req_001',
  filePath: './specs/requirements.pdf',
  description: 'Requirements specification document'
});

console.log(`Uploaded: ${artifact.filename}`);
console.log(`Size: ${artifact.size} bytes`);
console.log(`URL: ${artifact.downloadUrl}`);
```

### Node.js - Buffer

```typescript
// Upload from buffer (e.g., generated content)
const reportContent = await generateTestReport();

const artifact = await client.artifacts.upload({
  itemId: 'item_tc_001',
  content: Buffer.from(reportContent),
  filename: 'test-report.html',
  contentType: 'text/html',
  description: 'Automated test report'
});
```

### Node.js - Stream

```typescript
import { createReadStream } from 'fs';

// Upload from stream (memory efficient for large files)
const artifact = await client.artifacts.upload({
  itemId: 'item_req_001',
  content: createReadStream('./large-file.zip'),
  filename: 'archive.zip',
  contentType: 'application/zip'
});
```

### Browser - File Input

```typescript
// Handle file input in browser
const fileInput = document.getElementById('fileInput') as HTMLInputElement;

fileInput.addEventListener('change', async (event) => {
  const file = fileInput.files?.[0];
  if (!file) return;

  const artifact = await client.artifacts.upload({
    itemId: 'item_req_001',
    content: file,
    filename: file.name,
    contentType: file.type,
    description: 'User uploaded file'
  });

  console.log(`Uploaded: ${artifact.filename}`);
});
```

### Browser - Drag and Drop

```typescript
const dropZone = document.getElementById('dropZone');

dropZone.addEventListener('drop', async (event) => {
  event.preventDefault();

  const files = event.dataTransfer?.files;
  if (!files?.length) return;

  const uploads = Array.from(files).map(file =>
    client.artifacts.upload({
      itemId: 'item_req_001',
      content: file,
      filename: file.name,
      contentType: file.type
    })
  );

  const artifacts = await Promise.all(uploads);
  console.log(`Uploaded ${artifacts.length} files`);
});
```

### Upload to Project

```typescript
// Upload artifact at project level (not attached to specific item)
const artifact = await client.artifacts.upload({
  projectId: 'proj_abc123',
  filePath: './docs/architecture.png',
  description: 'System architecture diagram'
});
```

## Downloading Artifacts

### Node.js - Save to File

```typescript
import { writeFile } from 'fs/promises';

// Download artifact content
const content = await client.artifacts.download('art_abc123');

// Save to file
await writeFile('./downloads/requirements.pdf', content);
```

### Node.js - Stream to File

```typescript
import { createWriteStream } from 'fs';
import { pipeline } from 'stream/promises';

// Stream large files to avoid memory issues
const stream = await client.artifacts.downloadStream('art_abc123');
const output = createWriteStream('./downloads/large-file.zip');

await pipeline(stream, output);
```

### Browser - Download Link

```typescript
// Get download URL for browser
const url = await client.artifacts.getDownloadUrl('art_abc123', {
  expiresIn: 3600  // URL valid for 1 hour
});

// Create download link
const link = document.createElement('a');
link.href = url;
link.download = 'requirements.pdf';
link.click();
```

### Browser - Blob

```typescript
// Download as blob for in-browser processing
const blob = await client.artifacts.downloadBlob('art_abc123');

// Display image
const imageUrl = URL.createObjectURL(blob);
document.getElementById('preview').src = imageUrl;
```

## Listing Artifacts

### List Item Artifacts

```typescript
// List all artifacts for an item
const { data: artifacts } = await client.artifacts.list({
  itemId: 'item_req_001'
});

for (const artifact of artifacts) {
  console.log(`${artifact.filename} (${artifact.size} bytes)`);
  console.log(`  Uploaded: ${artifact.createdAt}`);
  console.log(`  Type: ${artifact.contentType}`);
}
```

### List Project Artifacts

```typescript
// List all artifacts in a project
const { data: artifacts } = await client.artifacts.list({
  projectId: 'proj_abc123',
  limit: 50,
  offset: 0
});
```

### Filter Artifacts

```typescript
// Filter by content type
const { data: pdfs } = await client.artifacts.list({
  projectId: 'proj_abc123',
  contentType: 'application/pdf'
});

// Filter by filename pattern
const { data: reports } = await client.artifacts.list({
  projectId: 'proj_abc123',
  filenameContains: 'report'
});

// Filter by date range
const { data: recent } = await client.artifacts.list({
  projectId: 'proj_abc123',
  createdAfter: new Date(Date.now() - 7 * 24 * 60 * 60 * 1000)  // Last 7 days
});
```

### Iterate All Artifacts

```typescript
// Iterate through all artifacts (handles pagination)
for await (const artifact of client.artifacts.iterate({
  projectId: 'proj_abc123'
})) {
  console.log(`${artifact.filename}: ${artifact.size} bytes`);
}
```

## Managing Artifacts

### Get Artifact Details

```typescript
const artifact = await client.artifacts.get('art_abc123');

console.log(`ID: ${artifact.id}`);
console.log(`Filename: ${artifact.filename}`);
console.log(`Size: ${artifact.size}`);

console.log(`Created: ${artifact.createdAt}`);
```

### Update Artifact Metadata

```typescript

  description: 'Updated description',
  metadata: {
    version: '2.0',
    reviewed: true
  }
});
```

### Delete Artifact

```typescript

await client.artifacts.delete('art_abc123');

// Delete multiple artifacts

  'art_abc123',
  'art_def456',
  'art_ghi789'
]);
```

### Move Artifact

```typescript

  targetItemId: 'item_req_002'
});
```

### Copy Artifact

```typescript
// Copy artifact to another item

  targetItemId: 'item_req_002'
});
```



## Progress Tracking

### Upload Progress

```typescript
const artifact = await client.artifacts.upload({
  itemId: 'item_req_001',
  filePath: './large-file.zip',
  onProgress: (progress) => {
    console.log(`Upload progress: ${progress.percent}%`);
    console.log(`${progress.loaded} / ${progress.total} bytes`);
  }
});
```

### Download Progress

```typescript
const content = await client.artifacts.download('art_abc123', {
  onProgress: (progress) => {
    console.log(`Download progress: ${progress.percent}%`);
  }
});
```

### React Progress Component

```tsx
import { useState } from 'react';

function FileUpload({ itemId }: { itemId: string }) {
  const [progress, setProgress] = useState(0);
  const [uploading, setUploading] = useState(false);

  const handleUpload = async (file: File) => {
    setUploading(true);
    setProgress(0);

    try {
      await client.artifacts.upload({
        itemId,
        content: file,
        filename: file.name,
        onProgress: (p) => setProgress(p.percent)
      });
    } finally {
      setUploading(false);
    }
  };

  return (
    <div>
      <input
        type="file"
        onChange={(e) => e.target.files?.[0] && handleUpload(e.target.files[0])}
        disabled={uploading}
      />
      {uploading && (
        <progress value={progress} max={100}>
          {progress}%
        </progress>
      )}
    </div>
  );
}
```

## Error Handling

```typescript
import {
  NotFoundError,
  ValidationError,
  FileTooLargeError,
  UnsupportedContentTypeError,
  QuotaExceededError
} from '@tracertm/sdk/errors';

try {
  const artifact = await client.artifacts.upload({
    itemId: 'item_req_001',
    filePath: './large-file.zip'
  });
} catch (error) {
  if (error instanceof FileTooLargeError) {
    console.log(`File too large: ${error.size} bytes (max: ${error.maxSize})`);
  } else if (error instanceof UnsupportedContentTypeError) {
    console.log(`Unsupported type: ${error.contentType}`);
  } else if (error instanceof QuotaExceededError) {
    console.log(`Storage quota exceeded: ${error.used}/${error.limit} bytes`);
  } else if (error instanceof NotFoundError) {
    console.log('Item not found');
  } else if (error instanceof ValidationError) {
    for (const err of error.errors) {
      console.log(`  ${err.field}: ${err.message}`);
    }
  }
}
```

## Best Practices

### 1. Use Streaming for Large Files

```typescript
// Bad - loads entire file into memory
const content = await fs.promises.readFile('./large-file.zip');
await client.artifacts.upload({ content, ... });

// Good - streams the file
await client.artifacts.upload({ filePath: './large-file.zip', ... });
```

### 2. Validate Before Upload

```typescript
import { stat } from 'fs/promises';

async function safeUpload(itemId: string, filePath: string) {
  const stats = await stat(filePath);

  const maxSize = 100 * 1024 * 1024;  // 100 MB
  if (stats.size > maxSize) {
    throw new Error(`File too large: ${filePath}`);
  }

  return client.artifacts.upload({ itemId, filePath });
}
```

### 3. Use Meaningful Descriptions

```typescript
// Good - descriptive
const artifact = await client.artifacts.upload({
  itemId: 'item_req_001',
  filePath: './specs/auth-requirements-v2.pdf',
  description: 'Authentication requirements specification v2.0 - includes OAuth 2.0 and SAML'
});
```
