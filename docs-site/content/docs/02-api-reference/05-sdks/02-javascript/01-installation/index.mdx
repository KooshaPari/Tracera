# Installation

Complete guide to installing the TraceRTM JavaScript SDK in your Node.js, browser, or TypeScript project.

## Requirements

- **Node.js**: 18.0 or higher (for server-side usage)
- **npm**: 8.0+ or **yarn**: 1.22+ or **pnpm**: 7.0+
- **TypeScript**: 4.7+ (optional, for TypeScript projects)

---

## Quick Install

### npm

```bash
npm install @tracertm/sdk
```

### yarn

```bash
yarn add @tracertm/sdk
```

### pnpm

```bash
pnpm add @tracertm/sdk
```

### Bun

```bash
bun add @tracertm/sdk
```

---

## TypeScript Support

The SDK is written in TypeScript and includes full type definitions. No additional `@types` packages are needed.

```typescript
import { TraceRTMClient } from "@tracertm/sdk";
import type { Item, Project, Link } from "@tracertm/sdk";

const client = new TraceRTMClient({
  apiKey: process.env.TRACERTM_API_KEY,
});

// Full type inference
const project: Project = await client.projects.get("proj_abc123");
const items: Item[] = (await client.items.list({ projectId: project.id })).data;
```

## Environment-Specific Installation

### Node.js Server

```bash
npm install @tracertm/sdk
```

```typescript
// CommonJS
const { TraceRTMClient } = require("@tracertm/sdk");

// ES Modules
import { TraceRTMClient } from "@tracertm/sdk";
```

### Browser (Bundler)

When using webpack, Vite, Rollup, or other bundlers:

```bash
npm install @tracertm/sdk
```

```typescript
import { TraceRTMClient } from "@tracertm/sdk";

// Use publishable key for browser
const client = new TraceRTMClient({
  apiKey: "rtm_pub_pk_...", // Public key only
});
```

### Browser (CDN)

```html
<script type="module">
  import { TraceRTMClient } from "https://cdn.jsdelivr.net/npm/@tracertm/sdk@latest/dist/browser.esm.js";

  const client = new TraceRTMClient({
    apiKey: "rtm_pub_pk_...",
  });
</script>
```

### Deno

```typescript
import { TraceRTMClient } from "npm:@tracertm/sdk";

const client = new TraceRTMClient({
  apiKey: Deno.env.get("TRACERTM_API_KEY"),
});
```

## Verify Installation

### Test Connection

```typescript
import { TraceRTMClient } from "@tracertm/sdk";

async function verifyInstallation() {
  const client = new TraceRTMClient({
    apiKey: process.env.TRACERTM_API_KEY,
  });

  try {
    const projects = await client.projects.list({ limit: 1 });
    console.log("Connection successful!");
    console.log(`Found ${projects.total} projects`);
  } catch (error) {
    console.error("Connection failed:", error.message);
  }
}

verifyInstallation();
```

### Check Version

```typescript
import { version } from "@tracertm/sdk";

console.log(`TraceRTM SDK version: ${version}`);
```

## Framework Integration

### Next.js

```typescript
// lib/tracertm.ts
import { TraceRTMClient } from "@tracertm/sdk";

// Server-side client (API routes, Server Components)
export const tracertm = new TraceRTMClient({
  apiKey: process.env.TRACERTM_API_KEY,
});

// app/api/projects/route.ts
import { tracertm } from "@/lib/tracertm";

export async function GET() {
  const projects = await tracertm.projects.list();
  return Response.json(projects);
}
```

### Express.js

```typescript
import express from "express";
import { TraceRTMClient } from "@tracertm/sdk";

const app = express();
const tracertm = new TraceRTMClient({
  apiKey: process.env.TRACERTM_API_KEY,
});

app.get("/api/projects", async (req, res) => {
  const projects = await tracertm.projects.list();
  res.json(projects);
});

app.listen(3000);
```

### Fastify

```typescript
import Fastify from "fastify";
import { TraceRTMClient } from "@tracertm/sdk";

const fastify = Fastify();
const tracertm = new TraceRTMClient({
  apiKey: process.env.TRACERTM_API_KEY,
});

fastify.decorate("tracertm", tracertm);

fastify.get("/projects", async () => {
  return fastify.tracertm.projects.list();
});

fastify.listen({ port: 3000 });
```

### NestJS

```typescript
// tracertm.module.ts
import { Module, Global } from "@nestjs/common";
import { TraceRTMClient } from "@tracertm/sdk";

@Global()
@Module({
  providers: [
    {
      provide: "TRACERTM_CLIENT",
      useFactory: () => {
        return new TraceRTMClient({
          apiKey: process.env.TRACERTM_API_KEY,
        });
      },
    },
  ],
  exports: ["TRACERTM_CLIENT"],
})
export class TraceRTMModule {}

// projects.service.ts
import { Injectable, Inject } from "@nestjs/common";
import { TraceRTMClient } from "@tracertm/sdk";

@Injectable()
export class ProjectsService {
  constructor(
    @Inject("TRACERTM_CLIENT")
    private readonly tracertm: TraceRTMClient
  ) {}

  async findAll() {
    return this.tracertm.projects.list();
  }
}
```

## Troubleshooting

### Module Resolution Issues

If you encounter "Cannot find module" errors:

```json
// tsconfig.json
{
  "compilerOptions": {
    "moduleResolution": "NodeNext",
    // or for older setups:
    "moduleResolution": "Node"
  }
}
```

### ESM/CJS Compatibility

For projects using CommonJS:

```javascript
// Use dynamic import
const { TraceRTMClient } = await import("@tracertm/sdk");

// Or configure package.json
{
  "type": "commonjs"
}
```

### Bundler Configuration

**Webpack:**
```javascript
// webpack.config.js
module.exports = {
  resolve: {
    extensions: [".ts", ".js"],
    mainFields: ["module", "main"],
  },
};
```

**Vite:**
```typescript
// vite.config.ts
export default {
  optimizeDeps: {
    include: ["@tracertm/sdk"],
  },
};
```
